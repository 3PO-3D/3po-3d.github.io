# Get CHRONOS

## What You Get

A single `.zip` file containing everything you need. Drop it in your Cinema 4D plugins folder and you're running.

---

## Requirements

| | |
|---|---|
| **Cinema 4D** | 2024 or later |
| **macOS** | 12 Monterey or later |

| **Python** | Included with Cinema 4D — nothing extra to install |

---

## Installation — 3 Steps

**Step 1.** Download `CHRONOS_v1.zip` from the link below.

**Step 2.** Unzip and copy the `CHRONOS` folder into your Cinema 4D plugins directory:
- **macOS:** `~/Library/Preferences/Maxon/Cinema 4D/plugins/`
- **Windows:** `%AppData%\Maxon\Cinema 4D\plugins\`

**Step 3.** Restart Cinema 4D. Open CHRONOS from **Plugins > CHRONOS > Open Dashboard**.

That's it. No Python setup. No additional dependencies.

## First Launch Walkthrough

1. Open Cinema 4D and load any scene (or start a new one)
2. Go to **Plugins > CHRONOS > Open Dashboard**
3. The CHRONOS panel opens — click **Activate** to connect it to your scene
4. Your first **Generation** is created automatically
5. Start adding Branches and Variables to your scene objects

No project file required. CHRONOS works with any `.c4d` file.
