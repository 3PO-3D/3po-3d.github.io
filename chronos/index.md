# C.H.R.O.N.O.S.

## Non-Destructive Scene Variation Management for Cinema 4D

> **One scene. Infinite variations. Zero destructive changes.**

CHRONOS is a Cinema 4D plugin that lets you manage multiple versions of your scene — lighting setups, material states, object configurations, camera angles — as named, switchable **Generations**, all stored in a single `.c4d` file.

No duplicating scenes. No "scene_v1_FINAL_REAL_v3.c4d". Just one clean file with full variation history built in.

## The Core Idea

Every parameter in your scene can exist in multiple states simultaneously. A **Generation** is a named snapshot of those states — switchable in real time, non-destructive, and inheritable.

Switch between a "Hero Lighting" generation and a "Product White" generation with a single click. Both live in the same file. Neither overwrites the other.
