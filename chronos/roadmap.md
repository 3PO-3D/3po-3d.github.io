# Roadmap

CHRONOS is built to grow. V1 ships the core variation system. Every release after it expands the reach of that system — into new workflows, new platforms, and new scales.

<div class="roadmap-list">

<div class="roadmap-item">
  <div class="version-badge active">
    <span class="v-number">V1</span>
    <span class="v-label">Live Now</span>
  </div>
  <div class="roadmap-content">
    <h3>Core Plugin</h3>
    <p>Everything you need to manage scene variations inside Cinema 4D.</p>
    <ul>
      <li>Named Generations — create, switch, delete, rename</li>
      <li>Dynamic Inheritance — live parent–child value chains</li>
      <li>Isolated Snapshots — sealed, frozen deliverables</li>
      <li>Branch and Variable system — track exactly what you choose</li>
      <li>Custom topology — flat, chain, or any tree shape</li>
      <li>Non-destructive write layer — zero scene corruption risk</li>
      <li>CHRONOS Dashboard — dockable panel inside Cinema 4D</li>
      <li>macOS and Windows support</li>
    </ul>
  </div>
</div>

<div class="roadmap-item">
  <div class="version-badge">
    <span class="v-number">V1.1</span>
    <span class="v-label">Next</span>
  </div>
  <div class="roadmap-content">
    <h3>Import / Export Presets</h3>
    <p>Share your setups. Move them between scenes.</p>
    <p>Export any Branch definition or full Generation set as a portable preset file. Import it into any project — same scene structure, same values, ready to use. Useful for studio-shared lighting rigs, standardised camera setups, and reusable material states across a product line.</p>
  </div>
</div>

<div class="roadmap-item">
  <div class="version-badge">
    <span class="v-number">V2</span>
    <span class="v-label">Upcoming</span>
  </div>
  <div class="roadmap-content">
    <h3>High-Performance Storage</h3>
    <p>Studio-scale projects. Instant response.</p>
    <p>CHRONOS V2 replaces the sidecar JSON with a structured database per project. Faster reads on large scenes, full query history, and the persistence backbone required for team workflows. Every Generation switch, every value edit — tracked and queryable. The foundation for everything in V3 and beyond.</p>
  </div>
</div>

<div class="roadmap-item">
  <div class="version-badge">
    <span class="v-number">V3</span>
    <span class="v-label">Upcoming</span>
  </div>
  <div class="roadmap-content">
    <h3>Standalone Dashboard + Omniverse Bridge</h3>
    <p>CHRONOS beyond Cinema 4D.</p>
    <p><strong>Standalone Dashboard</strong> — a native desktop application that connects to your Cinema 4D session. Manage your Generations from a separate window, on a second monitor, or completely independently of C4D. Full visual Generation tree with drag-to-reparent.</p>
    <p><strong>Omniverse / USD Bridge</strong> — every CHRONOS Generation becomes a USD Variant. Export your variation data as USD VariantSets — compatible with NVIDIA Omniverse, Houdini, Maya, Unreal Engine 5, and any USD-native pipeline. Your scene variations travel beyond Cinema 4D.</p>
  </div>
</div>

<div class="roadmap-item">
  <div class="version-badge">
    <span class="v-number">V4</span>
    <span class="v-label">Future</span>
  </div>
  <div class="roadmap-content">
    <h3>Plugin Ecosystem</h3>
    <p>Extend CHRONOS for your pipeline.</p>
    <p>A hot-pluggable addon system for custom CHRONOS extensions: macro engines, scene health checkers, batch render managers, and pipeline-specific integrations. Build once, load anywhere.</p>
  </div>
</div>

<div class="roadmap-item">
  <div class="version-badge">
    <span class="v-number">V5</span>
    <span class="v-label">Future</span>
  </div>
  <div class="roadmap-content">
    <h3>Studio Sync and Collaboration</h3>
    <p>Multi-seat. Multi-machine. Real-time.</p>
    <p>CHRONOS Generations as a shared studio resource. GitOps-compatible export for version-controlled workflows. Real-time multiplayer session sync — multiple artists in the same Generation tree simultaneously.</p>
  </div>
</div>

<div class="roadmap-item">
  <div class="version-badge">
    <span class="v-number">V6</span>
    <span class="v-label">Future</span>
  </div>
  <div class="roadmap-content">
    <h3>AI-Assisted Authoring</h3>
    <p>Describe a variation. CHRONOS creates it.</p>
    <p>An AI engine that understands your Generation structure, proposes new variations based on your existing data, and applies them through the standard CHRONOS write layer — with a diff preview before anything touches your scene. You approve. CHRONOS executes.</p>
  </div>
</div>

</div>

---

> CHRONOS is built on one idea: **your scene should never be held hostage by a variation workflow.**
>
> No duplicate files. No manual state management. No destructive changes. CHRONOS is infrastructure — always there, always non-destructive, always expanding.
